# `calendar.txt`

## 1. Descripción

Proyecto escrito en **Python**, que permite parsear, gestionar y manipular archivos de calendario personal en formato de texto plano (`calendar.txt`).

Permite leer, escribir, archivar días pasados y generar plantillas para días futuros. Está diseñado para ser simple, flexible y fácilmente integrable en flujos de trabajo basados en texto.

> **Nota:** El formato `calendar.txt` se encuentra documentado en la **Wiki** de este proyecto, aquí se documenta como funciona y cuales son sus características.

## 2. Pre requisitos

Debe tener instaladas en su máquina las siguientes herramientas:

- [python](https://www.python.org/): Versión 3.11 o superior.
- [uv](https://github.com/astral-sh/uv): Un administrador de proyectos y paquetes de Python extremadamente rápido, escrito en Rust.
